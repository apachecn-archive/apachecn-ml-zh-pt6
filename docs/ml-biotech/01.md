

# 第一部分：数据入门

本节介绍 Python、SQL 的基础知识，以及将原始数据转换为有意义的可视化和表示，作为强大数据科学项目的第一步。新手学生通常会发现自己被互联网或出版物上的大量数据科学内容所淹没。这本书通过关注在这个领域开始的最重要和最有价值的必须知道的元素来补救这个问题。
